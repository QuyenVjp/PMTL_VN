# Homepage Audit Reference

Source: `docs/design/pmtl-homepage-vercel-grade-audit.md`

Use that full document when the homepage or landing sections are being redesigned.

## Highest-value checks

- reduce surfaces to 3 primary personalities
- lock a predictable radius scale
- use gold strategically, not everywhere
- standardize card anatomy
- keep motion structural, not decorative
- tighten section rhythm
