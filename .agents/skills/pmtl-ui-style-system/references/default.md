# Default Variant

Use this when the user asks for PMTL UI without a special visual mode.

- Aim for warm editorial clarity with disciplined spacing.
- Favor asymmetric but controlled layouts over generic centered hero blocks.
- Use strong typographic hierarchy, restrained gold emphasis, and clean neutral surfaces.
- Prefer subtle shadows and soft borders only when hierarchy needs them.
- Avoid default 3-column feature grids when a more intentional composition is available.
- Build full UI states, not static happy-path screens.
