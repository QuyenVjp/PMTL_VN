# PMTL Creative Designer

Use this skill for the visual “why” of PMTL, not just the CSS “how”.

## Workflow

1. Read the target surface or section.
2. Decide the dominant need:
   - brand atmosphere
   - type and color direction
   - composition and spacing rhythm
   - motion tone
3. Read only the matching reference file.
4. Preserve PMTL’s quiet spiritual identity while keeping the UI contemporary and sharp.

## Read only what applies

- `references/brand-pillars.md`
- `references/typography-and-color.md`
- `references/layout-composition.md`
- `references/motion-tone.md`

## Design test

The output should feel:

- warm
- contemplative
- premium through discipline
- editorial without becoming antique
