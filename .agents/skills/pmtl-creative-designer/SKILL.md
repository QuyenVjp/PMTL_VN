# PMTL Creative Designer

Use this skill to transform a functional page into a premium, spiritual, and editorial experience.

## 1. Visual Philosophy: "Structure Felt, Not Seen"
- **Minimalist Boundaries:** Prefer negative space or subtle background tints over physical border lines.
- **Glassmorphism:** Use background-blur selectively for overlays to maintain depth without clutter.
- **Vertical Rhythm:** Ensure consistent spacing between sections. Large gaps create a sense of "calm."

## 2. Typography Mastery
- **Editorial Mix:** 
    - **Titles:** Use Elegant Serifs (e.g., Playfair Display or project default) for a spiritual, authoritative feel.
    - **Body:** Use high-legibility Sans-serifs (e.g., Inter, Geist) for utility.
- **Micro-typography:** Use `letter-spacing` and `line-height` to create "breathable" text. Never crowd the content.

## 3. The Gold Standard (Restrained Luxury)
- **Primary Emphasis:** Gold is for the "Divine" or "Most Important." It should be the single clearest path of action.
- **Restraint:** If everything is Gold, nothing is Gold. Use it for one CTA, one active nav item, or one key icon per view.
- **Neutral Palette:** Build the world in Stone, Sand, Earth, and Charcoal. Use Gold for the "Sun."

## 4. Premium "Linear-grade" Details
- **Micro-animations:** 
    - Subtle 2px "translate-y" upwards on hover.
    - Soft border-color transitions.
    - Skeletons that pulse with a gentle "spiritual" flow.
- **Information Density:** Organize complex data into clean, readable cards with a strict hierarchy (Eyebrow > Title > Description > Meta).

## 5. Design Checklist
When creating a design:
- Is the hierarchy obvious within 1 second?
- Does it feel "Sacred/Quiet" or "Busy/Techy"? (Aim for Sacred/Quiet).
- Are the gradients smooth and barely noticeable?
- Is there enough "white space" (or sand/stone space) to let the user breathe?
